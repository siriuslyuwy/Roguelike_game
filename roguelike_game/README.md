几何大战（Python/pygame）

项目总览（上下文快照/新对话可粘贴本段）
- 画面/帧率：1440×900，FPS=60；顶部面板高度=190。
- 战场：7 条战线、每线独立基地；摧毁对方≥4 座判胜。
- 模式：
- 正常模式（肉鸽 20 战）：开局选3兵种+1技能，随后进入节点式地图（参考《杀戮尖塔》）。路线以连线为准：只要完成一条连线上的任意节点，就能访问与之直接相连的上下层节点；已通关节点可再次移动经过（不再触发战斗），以便回溯到其他分支。战斗胜利后"兵种/技能/增益 3 选 1"，已解锁兵种在奖励选择中概率+50%；奖励金钱按战斗次数递增（100 起始，每场 +10），时间奖励：1分钟内+120金，2分钟内+60金，3分钟内+30金。单局兵种上限=7；AI 兵种数与出兵速度随关卡逐层提升。
  - 自由模式：开局无限配阵→自由选择增益层数→对战（AI 随机 7 兵种）。
- 操作：上/下 切战线；左/右 换兵；空格 出兵；Q/W/E 依序施放技能；Esc：对战内返回主菜单，主菜单按 Esc 退出程序。
- 视野：每线独立迷雾；可见边界=该线最前友军 x + 半屏。
- 顶部 HUD：
  - 第1行：资源 + 兵种栏（支持两行；槽位更宽、兵种名不截断；选中高亮边框更清晰）。
  - 第2行：技能与击杀资源（显示携带技能槽与所需击杀、当前击杀值），整体下移避免与兵种栏重叠。
  - 第3行：所选增益（名称 x 层数；自动换行），整体下移确保不重叠。
  - 战场可读性增强：所有单位头顶显示血条（低血变红）；单位图标叠加“功能标识”（远程短线、AOE点、治疗十字、光环细环、冲锋箭头、拦截弧线、刺客V、分裂线等）。

一、项目简介
一个横向对战的“几何体”战斗游戏：战场共有 7 条战线（自上而下 1-7 行）。左右两侧在每条战线上各有一座独立基地（共 7 座）。双方可从任意战线生产不同几何体士兵推进对战。支持“每线迷雾视野”“击杀值转化为技能资源（最多3槽）”“局外配阵”等玩法。先击破对方至少 4 座基地（多数）者胜。

二、核心玩法与规则
- 战线：共 7 条，单位仅与同一战线的目标交战。
- 基地：各有生命值；单位可对单位或敌方基地造成伤害。
- 生产：消耗资源生产单位；资源随时间增长；配阵限制单局可用兵种。
- 迷雾：每条战线独立计算视野。开局可见左半区；友军最前位置+半屏决定可见边界。
- 技能：每击杀 1 个敌方单位获得 1 点“击杀值”，自由模式最多携带 3 个技能，正常模式开局 1 个、奖励可解锁至 3 个；技能消耗各自的击杀值（默认 12/15/25/35 等），根据携带顺序对应 Q/W/E 施放。
- 单位（名称/定位已更新）：
  - 战士：均衡近战单体。
  - 盾卫：高生命、低伤、慢速近战。
  - 游击：近战单体，命中“近战单位”施加缴械1.8s并+0.3s攻冷；优先攻击“未被缴械”的近战；受远程伤害×1.3；冷却2.0s；费用90。
  - 狂战：近战AOE（小范围溅射）。
  - 牧师：远程治疗，无攻击。
  - 弓手：远程单体输出。
  - 法师：中远程AOE投射。
  - 犀牛：推线坦克，每次攻击造成AOE击退（力度随速度；当前为基础系数的2/3）；击退时自身同步前移等距，保持贴身。
  - 刺客：无视接敌持续前进，仅攻击远程/辅助类（治疗/光环/拦截）单位。
  - 破箭：无攻击；拦截半径内可拦截弹道，且50%概率以60%伤害反弹回对方方向（速度略降）。
  - 鼓手：光环移速/攻速增益（自身无输出）。
  - 矛兵：克制冲锋（对冲锋单位伤害×1.8并短暂打断）。
  - 冰弓：命中叠减速，满5层对同一目标触发一次性眩晕。
  - 自爆车：被击杀时对同线敌人造成范围伤害（当前半径40、伤害110），不再分裂。
  - 轻骑：仅“首次命中”触发AOE击退（力度较大），后续为单体；击退时自身同步前移等距。
- 胜负：任意一方有 ≥4 条战线的基地被摧毁（HP≤0）即判负；另一方胜。

三、主页与模式
- 主页提供：
  - 1：正常模式（肉鸽战役地图）
  - 2：百科（操作说明与兵种数值，支持滚动浏览：上下键/鼠标滚轮）
  - 3：自由模式（无限配阵 + 增益）

四、战役地图（肉鸽）流程
- 配阵：先在"战役配阵"阶段挑选 3 个兵种与 1 个技能（奖励可扩展至最多 3 技能，兵种最多 7 个）；商店可购买兵种/技能/增益（250金/件）。
- 地图结构：共 20 层节点，自上而下推进；节点类型含普通战斗、精英、事件、休整、商店与最终 Boss。每层随机生成在 5 条纵向轨道上，节点只会连接至临近轨道的前后层。
- 路线逻辑：选择第一层任意起点后开始旅程；清除一个节点即可解锁与其有连线的所有上下层节点，包含尚未走过的第一层节点（只要存在连线）。已通关节点可自由穿行用于换线，但不会再次进入战斗。
- 移动与天数：在地图上每移动一次视为过一天，界面右侧会实时显示“天数”与当前所处节点；当前节点与光标节点均有明显高亮。
- 奖励经济：首次战斗胜利获得 100 金，之后每场 +10；事件节点可额外获 50 金；时间奖励：1分钟内通关+120金，2分钟内+60金，3分钟内+30金；休整节点会将己方基地 HP 恢复至满值。
- 奖励选择：战斗胜利后"兵种/技能/增益 3 选 1"，已解锁兵种在奖励选择中概率+50%，更容易获得高级兵种。
- 操作：左右/上下键在可达节点之间切换，Enter/Space 进入；PgUp/PgDn 或鼠标滚轮滚动地图；商店中 R 键刷新商品（-50金）；Esc 返回主菜单。
- 战斗难度：AI 兵种池、出兵间隔与基础 HP 倍率随层数与节点类型递增（精英/Boss 调高 1-2 档）；每过一天敌军伤害+5%，生命+3%。

五、操作说明（玩家=左侧，右侧为简单 AI）
- 上/下：选择当前生产战线。
- 左/右：切换兵种（按当前可用兵种循环）。
- 空格：在当前战线出兵（若资源足够）。
- Q：释放所选技能（若有充能）。
- Esc：对战内返回主菜单；主菜单按 Esc 退出程序（百科中返回主页）。

六、运行环境
- Python 3.9+
- pygame 2.5+

七、安装与运行
1) 安装依赖：
```bash
pip install -r requirements.txt
```
2) 启动游戏：
```bash
python main.py
```

十二、中文字体（文字显示为“口口口/框框”的解决办法）
- 已集成字体加载器，优先加载 `assets/fonts` 中的中文字体，其次尝试系统字体（微软雅黑/黑体/等线/宋体等）。
- 如仍出现方框：
  1) 将支持中文的字体文件放入 `assets/fonts/`，例如 `NotoSansSC-Regular.otf` 或 `msyh.ttc`；
  2) 重新运行游戏。


十三、特殊技能（击杀值消耗）
- 选择与携带：
  - 正常模式：配阵阶段先选 1 个技能开局，通关后可在奖励中追加技能，最多 3 个。
  - 自由模式：配阵阶段可直接勾选最多 3 个技能。
  - 携带顺序决定技能槽位，对应键位 Q/W/E。
- 击杀值：击杀敌方单位获得 1 点击杀值，可累积并按技能消耗使用；增益“战术超频”可降低消耗。
- 施放：按 Q/W/E 触发对应技能，若击杀值不足则无法施放（自由模式可启用“无限技能”用于练习/调试）。
- HUD：顶部第 2 行显示技能槽（键位/名称/需求/是否就绪）及当前可用击杀值。
- 技能类型（示例，实际以游戏内为准）：
  - 全线出兵：瞬时在所有战线各生产 1 次当前选中兵种（无资源开销）。
  - 冰霜射线：指定战线敌军眩晕 4 秒。
  - 死亡射线：指定战线敌军瞬间清除。
  - 轰轰轰：对指定战线投下多段范围爆炸。
  - 守护：指定战线友军获得护盾。
- 约束与判定：
  - 技能仍受迷雾与配阵限制（如全线出兵只会召唤在可见战线且遵守兵种限制）。
  - 技能效果数值、消耗倍率、持续时间等可在 `game/constants.py` / `game/game.py` 调整。

十四、增益系统（局外成长）
- 获取时机：
  - 正常模式：每关胜利后“兵种/增益 3 选 1”。
  - 自由模式：开局自由选择增益及层数上限后进入对战。
- 展示与叠加：顶部第 3 行展示“增益名称 × 层数”，自动换行不遮挡。
- 增益类型（示例）：资源增长提升、单位生命/伤害/攻速加成、特定兵种强化、击退强度微调等。实际以游戏内为准。
- 配置入口：增益候选池、数值范围与层数上限可在 `game/constants.py` 调整。

十五、百科内容
- 可在主菜单进入“百科”，支持上下键/鼠标滚轮滚动浏览。
- 内容覆盖：
  - 操作说明与热键（含技能 Q/W/E、切线/换兵/出兵/返回）。
  - 兵种基础数值与机制标签（远程/治疗/光环/冲锋/拦截/刺客/AOE/缴械/击退等）。
  - 特殊技能的击杀消耗与效果简介。
  - 增益列表与叠加说明（如已解锁/可见）。

十六、特殊机制支持
- 无敌：带有该标签的单位不会损失生命值，伤害会被护盾完全抹除。
- 伤害反弹：单位受到的实际伤害会百分百回击到来源（含近战与投射物）。
- 吸血：按造成的有效伤害比例回复自身生命值，超出部分不上溢。
- 穿透：远程投射物自带穿透段数，命中首个目标后可继续打击后列。
- 点燃道路：命中时生成持续燃烧区域，对经过的敌方单位造成每秒伤害（UI 以橙/紫火焰提示）。
- 隐身：不会被敌方锁定或阻挡，可被范围/误伤波及；战场中以半透明显示。
- 控制免疫：免疫减速、眩晕、禁锢等一切控制效果。

十七、兵种升级机制
- 兵种可升至Lv4，升级会提升全属性（HP、伤害、速度、射程等）并降低冷却时间
- 2级：全属性+5%（相对于基础值）
- 3级：全属性+10%（相对于基础值，每级+5%）
- 4级：保持原计算方式（约+9%），并解锁满级特效

十八、满级兵种特效（Lv4）
- 战士：首次受到伤害后立即进入 5 秒无敌，期间完全免伤。
- 盾卫：被击中时反弹 20% 实际伤害给攻击者。
- 游击：近战命中近战单位时，眩晕效果改为小范围 AOE。
- 狂战：造成伤害的 20% 转化为自我治疗（吸血）。
- 牧师：治疗改为光环式 AOE，对身边缺血友军同时回复。
- 弓手：自带 1 段穿透，后续目标伤害保留 70%。
- 法师：命中时点燃地面 3 秒（火焰不叠加，但会刷新持续时间）。
- 犀牛：同一目标累计被击退 4 次后额外眩晕 1 秒。
- 刺客：停留在己方半场时进入隐身，敌方无法以其为目标。
- 破箭：成功反弹投射物时自我恢复少量生命值。
- 鼓手：光环范围内全友军每 10 秒获得一次 25% 最大生命护盾（持续 5 秒）。
- 矛兵：获得完全控制免疫（免疫减速、眩晕、禁锢等）。
- 冰弓：触发眩晕时，连同目标周围敌军一并眩晕。
- 自爆车：爆炸后在原地点燃道路，持续 5 秒灼烧敌军。
- 轻骑：冲锋结束后若 2 秒内未攻击，将再次获得一次性的冲锋 AOE 效果。
