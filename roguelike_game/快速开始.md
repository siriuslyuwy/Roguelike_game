# 🚀 精灵图快速开始指南

## ✅ 好消息！

你的战士精灵图已经可以在游戏中显示了！

## 🎮 立即测试（三步走）

### 步骤 1: 运行调试工具
```bash
py sprite_debug.py
```

你会看到：
- 战士的 idle_0.png 精灵图
- 实时动画播放
- 各种尺寸的预览

**操作提示:**
- ← → : 切换单位
- 1/2/3 : 切换动画
- ESC : 退出

---

### 步骤 2: 在游戏中查看
```bash
py main.py
```

1. 按 **3** - 进入自由模式
2. 按 **空格** - 选择战士
3. 按 **Tab** - 切换到技能
4. 按 **空格** - 选择技能
5. 按 **回车** - 开始战斗
6. 按 **空格** - 召唤战士

你会在游戏中看到战士的精灵图！

---

### 步骤 3: 添加更多动画帧

在 `assets/sprites/warrior/` 文件夹中添加：

**让待机动画更流畅:**
- `idle_1.png` - 轻微变化
- `idle_2.png` - (可选) 回到初始状态

**添加行走动画:**
- `walk_0.png` - 左脚前
- `walk_1.png` - 过渡
- `walk_2.png` - 右脚前
- `walk_3.png` - 过渡

**添加攻击动画:**
- `attack_0.png` - 准备姿态
- `attack_1.png` - 挥击中
- `attack_2.png` - 收招

添加后，再次运行调试工具查看效果！
