# 音乐与音效素材要求

为了让游戏获得最佳的听觉体验，请按照以下规范准备音频文件并放入本文件夹（`assets/music`）。

## 1. 背景音乐 (BGM)

游戏支持根据敌方单位数量切换不同节奏的 BGM。
**现在只需提供基础版，游戏会自动生成加速版！**

### 文件命名与功能
*   **`bgm_base.mp3`** (必须): 基础背景音乐。
*   `bgm_fast.mp3` (可选): 如果提供了，优先使用这个；否则由 base 自动生成 1.15x 加速版。
*   `bgm_intense.mp3` (可选): 如果提供了，优先使用这个；否则由 base 自动生成 1.35x 加速版。

**注意**:
*   请截取好音频的循环段落，尽量保证首尾相接时听感自然。
*   自动加速会稍微提高音调（更紧张），不需要手动处理。

## 2. 音效 (SFX)

击杀音效支持连击反馈（击杀越多，音效越重）。

### 文件命名
*   **`kill_light.mp3`**: 普通击杀音效（单次击杀）。
*   **`kill_medium.mp3`** (可选): 连续击杀音效。
*   **`kill_heavy.mp3`** (可选): 大量连续击杀音效。

**注意**:
*   如果没有提供可选文件，所有击杀都将播放 `kill_light.mp3`。
*   音效文件应尽可能短（0.5秒以内），且开头不要有静音空白，以保证打击感的即时性。

## 3. 其他说明
*   格式推荐: `mp3` 或 `ogg`。
*   请确保文件名完全一致（小写）。
